## :arrow_down: Install

- Install with [yarn](https://yarnpkg.com/)

```bash
yarn add prettier @lucasbernardol/prettier-config -D
```

- Usage/create `prettier.config.js`

```bash
echo "module.exports = require('@lucasbernardol/prettier-config');" > prettier.config.js
```
